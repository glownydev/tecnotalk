Bienvenue sur l'API Tecnotalk

Cette API fournit les endpoints suivants :

Authentification

POST /api/users/register - Inscription d'un nouvel utilisateur

POST /api/users/login - Connexion d'un utilisateur

Classes

GET /api/classes - Liste des classes

POST /api/classes - Création d'une classe

GET /api/classes/:id - Détails d'une classe

PUT /api/classes/:id - Modification d'une classe

DELETE /api/classes/:id - Suppression d'une classe

Élèves

GET /api/students - Liste des élèves

POST /api/students - Création d'un élève

GET /api/students/:id - Détails d'un élève

PUT /api/students/:id - Modification d'un élève

DELETE /api/students/:id - Suppression d'un élève

Événements

GET /api/events - Liste des événements

POST /api/events - Création d'un événement

GET /api/events/:id - Détails d'un événement

PUT /api/events/:id - Modification d'un événement

DELETE /api/events/:id - Suppression d'un événement
